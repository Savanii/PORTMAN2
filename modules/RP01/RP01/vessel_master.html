Download CSV Template

2Fill in the data

  • The template's second row shows the expected format for each column — keep it or delete it, it is ignored on upload.
  • One complete row per vessel call. Do not include total rows — any row without a Vessel Name is ignored.
  • All timings: DD-MM-YYYY HH:MM (e.g. 25-12-2024 07:48). A dash (-), blank or #N/A/#DIV/0! is treated as empty.
  • KPI columns (waiting / working / movement times) are in days, as in the existing sheet. Numbers may include commas.

3Upload & replace

This deletes ALL existing vessel call master rows and inserts the file contents. Nothing is changed if the file has errors.
